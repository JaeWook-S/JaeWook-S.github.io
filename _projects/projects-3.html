Project Overview

This research analyzes cellular state transitions in cancer cells using Immunogenic Cell Death (ICD), an immunotherapy approach. ICD-treated cells undergo a stepwise progression from normal cells → swollen cells (S.C) → dead cells (D.C).

Compared to conventional apoptosis, ICD exhibits more clearly distinguishable cellular state changes, showing a characteristic crossover pattern between S.C and D.C populations over time.

Cell states are identified using a combination of DIC images and GFP/RFP fluorescence images. GFP highlights swollen cells, while RFP indicates dead cells, enabling state classification.

However, acquiring and aligning DIC, GFP, and RFP images requires long-term experiments of up to 48 hours, making the workflow time-consuming and inefficient.

Therefore, this research aims to generate GFP and RFP fluorescence images directly from DIC images and use them to automatically classify swollen (S.C) and dead (D.C) cells.

Method

Data Preprocessing

I inspected the dataset quality and found that some images were severely affected by strong light reflections, making the cells barely distinguishable. These images were excluded from the dataset.

In addition, for RFP images, training the model with raw data resulted in nearly black output images. Further analysis indicated that the red fluorescence signal in RFP images was relatively weak, preventing the model from learning meaningful features. To address this issue, I applied brightness enhancement as a preprocessing step to improve the visibility of fluorescent signals.

Since the dataset was limited to approximately 400 images, I applied K-fold cross validation to improve generalization performance, with K set to 5.

Result

GUI

The goal of this study was to develop a GUI-based web system to enable more efficient analysis of DIC images collected from a cell biology laboratory.

On the first page, users can input basic experimental information and select the target cell type for analysis. The system supports SC, DC, or both cell types depending on the experimental setup.

After clicking the Start button, model training and the detection pipeline are automatically executed using the uploaded data, and the results are forwarded to the second page.

The second page displays images organized in a plate-based format for easier inspection. When a specific image is selected, the system navigates to the third page, where all images belonging to the corresponding plate can be analyzed in detail. This includes SC/DC classification results, generated images, detection outputs, and temporal trend graphs.

This GUI system is currently being used in actual cell biology laboratory experiments and has been applied to real-world data analysis workflows.
