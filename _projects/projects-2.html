Project Overview

I am conducting an autonomous driving project using the F1TENTH platform as part of a reinforcement learning course.

The baseline demo is based on a DQN, where the state consists of 1080-dimensional LiDAR distance data covering a 270-degree field of view, along with the vehicle’s dynamic information. The action space is composed of five discrete steering commands: sharp left, left, straight, right, and sharp right.

In this study, I aim to improve the policy performance based on this baseline and explore reinforcement learning methods that enhance both driving speed and stability.
